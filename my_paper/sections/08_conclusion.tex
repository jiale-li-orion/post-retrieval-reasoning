\section{Conclusion}
\label{sec:conclusion}

Long-horizon agents need more than compressed memories.
They need runtime access interfaces that match artifact usage semantics and evidence views that expose the roles required by current decisions.
This paper frames that requirement as \methodname{}, separating artifact storage, typed access primitives, primitive traces, evidence-view construction, and downstream decoding.

The final version will use ATM-Bench and STALE experiments to determine how far this decomposition explains real memory-use failures and whether role-aware evidence views or state-validity primitive stacks improve downstream decisions.
